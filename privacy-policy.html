Privacy Policy

Tree Tabs  ·  by Raj Golawar  ·  Last updated: June 2026

🔒

Short version: Tree Tabs collects no personal data, sends nothing to any server, and stores only the information needed to remember your tab tree and settings — entirely on your own device. No accounts. No tracking. No ads.

📋 Overview

Tree Tabs is a Chrome browser extension that organises your open tabs into a collapsible, searchable tree structure inside Chrome's side panel. This policy explains what data the extension accesses, why, and how it is handled.

📂 Data We Access

Tree Tabs accesses only what is necessary to function. Nothing is ever transmitted off your device.

DataWhyStoredShared
Tab titles, URLs, favicons To display the tab tree and build parent–child relationships Locally on device Never
Tab opener relationships To automatically nest child tabs under the tab that opened them Locally on device Never
User settings
theme, indent, language, etc.		 To remember your preferences across sessions Locally on device Never

🚫 Data We Do NOT Collect

  • Personal information (name, email, account details)
  • Browsing history beyond your current open tabs
  • Analytics, telemetry, or crash reports
  • Cookies or tracking identifiers of any kind
  • Data transmitted to any external server — ever

🔑 Permissions Explained

Every permission is limited to a specific feature. No permission is requested speculatively.

PermissionWhy it is required
tabs List, monitor, activate, create, and close tabs — the core function of the extension
storage Save the tab tree structure and user settings locally in the browser via chrome.storage.local
sidePanel Render the tree UI inside Chrome's native side panel
tabGroups Create, label, and remove Chrome Tab Groups when using Group by domain and Ungroup features
contextMenus Show a right-click context menu on tab items (Switch To, Close, Duplicate, etc.)

💾 Local Storage

All data persisted by Tree Tabs is stored using chrome.storage.local — a sandboxed, on-device store that is private to the extension and never transmitted anywhere.

You can delete all stored data at any time: open chrome://extensions → find Tree Tabs → click Details → click Clear data. Alternatively, removing the extension deletes all its stored data immediately.

🤝 Third-Party Services

Tree Tabs uses no third-party services, SDKs, analytics platforms, advertising networks, or external APIs of any kind. There are no requests to any server outside of Chrome's built-in extension APIs.

👶 Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect any information from them.
